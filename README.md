# Clario

Minimal full-stack hackathon starter for **Clario** (AI file organizer).

## Project Structure

- `frontend/` - React + Vite UI
- `backend/` - Node.js + Express API
- `ai/` - AI prompt and analyzer stubs

## Run Locally

### Frontend

```bash
cd frontend
npm run dev
```

### Backend

```bash
cd backend
node server.js
```

## Notes

- Purposefully minimal setup for quick hacking.
- No database, auth, or deployment config included.
